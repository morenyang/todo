# vue-todo-list

> A simplify todo list built by vue.js

这个仓库只用来展示todo app, 可能会稍微落后于源码仓库的版本

### 关于使用
- 单击列表中的事件可以添加星标
- 桌面端中将鼠标移到事件上后会出现删除按钮
- 移动端中点击 Edit/Done 会出现/隐藏删除按钮
